# 탭 1 : 연락처 구축

먼저 기본적으로 저희 팀은 탭 1, 2, 3 간의 정보를 서로 이용하고자 했습니다. 탭 1에서 편집한 연락처들을 통해 탭 3에서 팀 매칭을 진행하게 됩니다!

주소록에서는 휴지통을 눌러 정보의 삭제가 가능합니다. 또한 이미지 아이콘을 눌러 신상정보의 편집을 할 수 있습니다.

로그아웃 버튼을 누르면 탭 0으로 돌아갑니다.
